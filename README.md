# General Projects

A collection of diverse software engineering projects, featuring full-stack web applications and low-level operating system implementations.

## Projects

### Web Applications

- **Helpdesk System** - A ticketing and support management system for handling customer service requests
- **HR Form Request System** - A comprehensive human resources form submission and tracking system with separate backend and frontend components

### OS-Level Implementations

- **Custom OS Scheduler** - A RISC-V based operating system kernel with advanced process scheduling algorithms including priority-based and deadline-based scheduling
- **Search Engine Clone** - A distributed search engine implementation featuring web crawling, page ranking, and compressed sparse row (CSR) data structures for efficient indexing

## Overview

This repository showcases both high-level application development and systems-level programming, demonstrating proficiency in full-stack web development, kernel programming, and algorithms.
